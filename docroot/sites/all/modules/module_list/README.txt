Module List
============

Description
-----------

Hi drupallers,
Tired of knowing the list of custom modules in the site? Module list will
help you to list out the custom and contributed module list available
 in the site.

Installation
------------
1. Extract the module in the folder sites/all/modules/contrib.

2. Go to the Module page at Administer > Modules
 (http://example.com/admin/modules) and enable it.

To view the list please go to Administer > Reports > Module list or
  Modulesadmin/reports/module-list.

Download and Install using Drush:
drush dl module_list - To download the module.
drush en module_list - to enable the module.
